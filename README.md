# AI Chat Interface 🤖✨

A beautiful, fully client-side AI Chat Interface built with pure HTML, CSS, and JavaScript. 

**[🌟 View Live Demo Here](https://shivani-vishwakarmaa.github.io/AI-chat-interface-/)**

---

## 🎨 Features
- **Fast & Lightweight:** Zero frameworks, no build tools, no backend required.
- **Multi-Conversation Support:** Create, switch between, and manage multiple chat threads easily.
- **Message Persistence:** Your conversations are automatically saved in the browser.
- **Markdown Support:** Renders headings, bold text, code blocks, tables, and more.
- **Dark/Light Mode:** One-click theme toggle that remembers your preference.
- **Animated UI:** Smooth typing indicators, spring-eased bubble animations, and avatar pop-ins.
- **Mobile Responsive:** Adapts beautifully to all screen sizes.

## 🛠️ Project Structure
- `index.html` — App layout and structure
- `style.css` — Styling, themes, and animations
- `script.js` — Core application logic and state management

## 🚀 Getting Started Locally

Getting this project running locally is extremely simple:

1. **Download or clone** the repository.
2. **Open `index.html`** in any modern web browser.
3. Start typing in the chat!

*Tip: You can also use tools like VS Code Live Server for a better development experience.*

## ⚙️ Connecting a Real AI API
Currently, the chat interface simulates AI responses for demonstration purposes. If you'd like to use a real AI like ChatGPT or Claude:
1. Open `script.js`
2. Locate the `pickAIResponse()` function
3. Replace the simulated logic with a simple `fetch()` call to your API of choice!
